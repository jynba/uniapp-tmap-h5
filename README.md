# uniapp-tmap-h5

uniapp 开发 h5 的 map 组件，使用 tmap

## 前置步骤

下载 node
npm i pnpm

## 使用步骤：

1. git clone https://github.com/jynba/uniapp-tmap-h5.git
2. cd uniapp-tmap-h5
3. pnpm i
4. pnpm dev:h5
